# Camolyt_-multi-camera-live-streaming
CamoLyt is a browser-based multi-camera live streaming platform. 
<br>
It allows multiple devices(like phone and laptop) to act as camera sources in a single session. 
<br>
Designed for workshops, classrooms, and small-scale broadcasts. 
<br>
Eliminates the need for expensive hardware or complex setup. 
<br>
Built using modern web technologies like WebRTC.
